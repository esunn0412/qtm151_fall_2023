(d) Create a function with four inputs $f(y,x,b0,b1)$

$\qquad e2 = (y - b_0 - b_1 * x)^2$

  • The function should return the mean of e2
  • This is the mean-square error (mse)
  • Call the function "fn_mse"
  • Let $y$ be the column "Potential"
  • Let $x$ be the column "Age"
  • Does
    $\qquad (b_0 = 80, b_1 = 0.4)$ or
    $\qquad (b_0 = 80, b_1 =-0.3)$ or
    $\qquad (b_0 = 80, b_1 = -0.5)$
    produce the lowest value of mse?
  • Print your results
In[]:
def fn_mse (y, x, b0, b1):
    e2 = (y-b0-b1*x)**2
    return np.mean(e2)

print(fn_mse(dataset["Potential"], dataset["Age"], 80, 0.4))
print(fn_mse(dataset["Potential"], dataset["Age"], 80, -0.3))
print(fn_mse(dataset["Potential"], dataset["Age"], 80, -0.5))

list_status = [0,1]
list_prob = [0.5, 0.5]
n = len(dataset)
np.random.seed(42)
dataset["status"] = np.random.choice(list_status,
                                     size = n, 
                                     p = list_prob)

data0 = dataset.query("status == 0")
data1 = dataset.query("status == 1")
